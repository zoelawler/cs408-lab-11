# Dynamic Table

## Overview

A simple inventory table the dynamically updates when adding or deleting from it. It stores the item id, name, and price. The table initially shows as empty, but once you click "Load Data" the information will be loaded. It is connected to DynamoDB where you can see how the data changes.
